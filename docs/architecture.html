Frontend

React + Vite on Vercel

  • fadejunkie.com — main app
  • merch.fadejunkie.com — separate deploy
  • Vercel Edge serves static assets globally
  • React Router for client-side routing

Auth + Identity

Clerk + Google OAuth

  • Clerk handles the full OAuth handshake
  • Google as identity provider — one-tap sign-in
  • Clerk issues short-lived JWTs to the app
  • Convex verifies JWTs via Clerk's JWKS endpoint

Backend + Data (planned)

Convex + Redis

  • Convex — isolated deployment, not shared
  • Real-time subscriptions drive Group Mode
  • 6 tables: users, progress, sessions, rooms, players, answers
  • Redis — room state + leaderboard hot cache (Phase 2+)
